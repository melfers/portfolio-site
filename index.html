Nom List

Nom List is an app that lets users search for restaurants in their area, and create lists of restaurants they'd like to visit, have already visited, or any other category they'd like to keep track of.

Nom List was built with:
  • React
  • Redux
  • JavaScript
  • Node
  • MongoDB

Green Thumb

Green Thumb is an app that lets users log notes and watering dates on their houseplants so that they can track when they need to water the plant next.

Green Thumb was built with:
  • JavaScript
  • jQuery
  • HTML / CSS
  • Node
  • MongoDB

Season was built with:
