Projects

Nom List

Nom List is an app that lets users search for restaurants in their area, and create lists of restaurants they'd like to visit, for tracking purposes. For the foodie who can't keep up with all the hot new restaurants popping up, Nom List allows users to keep track of where they've been, where they want to go, and any other list they can think up.

Nom List was built with:

  • React
  • Redux
  • JavaScript
  • Node
  • MongoDB

Green Thumb

Green Thumb is an app that lets users log notes and watering dates on their houseplants so that they can track when they need to water the plant next. For many plant owners, it can be hard to remember when they last watered their various houseplants, or at what frequency a certain plant needs watering. Green Thumb is the solution to tracking watering dates to be sure that your plant stays happy and healthy.

Green Thumb was built with:

  • JavaScript
  • jQuery
  • HTML / CSS
  • Node
  • MongoDB

Season

Season is an app that searches for recipes based on produce that is in-season. The user selects a month that they'd like to find in-season produce for, and is then shown a list of items that are available for the selected month. After selecting one or more items, the user is shown recipes that have the searched items as ingredients.

Season was built with:

  • JavaScript
  • jQuery
  • HTML
  • CSS
